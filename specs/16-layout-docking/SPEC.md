# Modul 16: Responsive Layout & Window-Management

## Breakpoints
| Modus | Breite | Layout |
|-------|--------|--------|
| Voll | >1400px | Alles nebeneinander |
| Mittel | 700-1400px | Stacking, FX als Overlay, Tabs |
| Kompakt | <700px | Schmale Strips, Tab-Navigation |

## Features
- Proportionale Skalierung aller Controls
- Keine fixe Mindestgröße
- Touch: Größere Hitboxen
- Layout-Präferenz in Settings
- Multi-Monitor: Sektionen verschiebbar (Zukunft)
- Dock-Handles an jedem Panel (6 Dots, Drag vorbereitet)
- Tiling-WM Kompatibilität (i3, Sway, Hyprland)
